## 🚀 QUICK START COMMANDS

Start Backend:
  $ cd backend
  $ mvn clean install
  $ mvn spring-boot:run
  → Available at: http://localhost:8080

Start Frontend:
  $ cd frontend
  $ python3 -m http.server 8000
  → Available at: http://localhost:8000

## 📡 API ENDPOINTS

GET  /api/locations
     Returns: List of 550+ global locations

GET  /api/locations/{location}/encoding
     Returns: Numeric encoding for location

POST /api/predict
     Request:  PredictionInput (JSON)
     Response: PredictionResult (JSON)

GET  /api/health
     Returns: Service status
