🚀 QUICK START

1. Start Backend:
   $ cd backend
   $ mvn spring-boot:run
   → http://localhost:8080

2. Start Frontend:
   $ cd frontend
   $ python3 -m http.server 8000
   → http://localhost:8000

3. Open browser to http://localhost:8000

🔧 TROUBLESHOOTING

Backend won't start:
  → Check Java 11 installed: java -version
  → Check Maven installed: mvn -version
  → Clean rebuild: mvn clean install -DskipTests

Frontend not connecting:
  → Verify backend running on :8080
  → Check browser console (F12)
  → Update API URL in frontend/src/js/api.js

Frontend not loading:
  → Ensure server running
  → Clear browser cache (Ctrl+F5)
  → Check file paths
