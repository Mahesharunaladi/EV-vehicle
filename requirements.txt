# Installation Instructions
# ==========================
# 
# Option 1: Using Maven (Recommended)
# ------------------------------------
# Navigate to the project directory and run:
#   mvn clean install
#
# This will automatically download all dependencies defined in pom.xml
#
# Option 2: Build and Run
# ------------------------
#   mvn clean package        # Build the JAR file
#   java -jar target/ev-demand-prediction-1.0.0.jar  # Run the application
#
# Option 3: Development Mode
# ----------------------------
#   mvn spring-boot:run      # Run directly using Spring Boot Maven plugin
#
# System Requirements
# ====================
# - Java 11+ installed and in PATH
# - 500MB+ RAM for development
# - 200MB+ disk space for Maven cache and dependencies
# - Internet connection for initial dependency download
#
# Troubleshooting
# ================
# If Maven fails to download dependencies:
#   1. Check your internet connection
#   2. Clear Maven cache: mvn clean
#   3. Try with offline mode: mvn -o install (if dependencies were previously downloaded)
#   4. Force update of dependencies: mvn clean install -U
